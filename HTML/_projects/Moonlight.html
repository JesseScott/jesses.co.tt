Moonlight



Moonlight is an iOS & Android application that I made as an experiment to learn AngularJS and PhoneGap. Based off of an orignal idea by an acquanitance Kelly Andres - who was a co-resident during the MARIN Residency where I made Sonic Drift , the implementation is very simple - cast a glow across the screen based off of the current phase of the moon.

So that's what the app does - queries the lunar cycle, thanks to the SunCalc library - then uses a HTML5 Canvas to light up the screen.
And that's all it does.

The Android version is available here and the iOS version is pending approval.

The source code can be found on my GitHub.

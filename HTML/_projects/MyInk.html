MyInk



MyInk is an iOS application that I made while working at a mobile development agency.
It allows you to capture handwritten fonts, and through a custom keyboard extension for iOS, transform any written text into a message that can be sent over SMS, Email, or any social platform.

The application is currently undergoing a Beta testing phase, and will be avaliable to the public soon.

Source code is not available, as this is a client project.

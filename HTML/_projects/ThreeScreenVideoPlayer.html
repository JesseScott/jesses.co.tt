Three Screen Video Player

A simple three screen video player with playback control (for use with Matrox TH2GO)
that I made for my friend Zac Rothman, to use for a performative installation at Brief Encounters.

Made with Processing and using the libraries GL Graphics, GS Video, and ControlP5.

The source code for the project is available on my GitHub.
